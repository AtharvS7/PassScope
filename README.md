# PassScope – Password Strength Analyzer

PassScope is a lightweight client-side password strength analyzer that evaluates password security in real time. The tool calculates password strength using heuristics such as length, character diversity, entropy estimation, and common password detection.

## Features

- Real-time password strength meter
- Security score from 0–100
- Estimated brute-force crack time
- Hack probability estimate
- Password improvement suggestions
- Entropy calculation
- Transparent scoring formula explanation
- Copy password to clipboard
- Password show/hide toggle

## Tech Stack

- HTML5
- CSS3 (with CSS variables for theming)
- Vanilla JavaScript (no frameworks or libraries)

## Privacy

All password analysis happens locally in the browser.
No passwords are stored, transmitted, or shared with any third party.
Your data never leaves your device.
